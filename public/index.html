MEAN Stack

MongoDB, Express, AngularJS, Node.js

See it live or on GitHub.

A production quality Node server ready to extend http(s) routes and your choice of authentication with ease. Paired with a responsive Angular front-end, this project demonstrates REST requests factored out as a service to save on repetitive coding (and mistakes). All a developer needs to do is dependency inject and name the route to match a supported server side path and BOOM, access to MongoDB automatically implementing your choice of security.

Clean server and client-side tests using Mocha/Chai and Jasmine/Karma respectively make for a quick catch of any breaking code. Oh, and just in case fast page loading and minimal file size is on the list of concerns, the build process compresses all the client-side JavaScript into a single bundled file for fast network transfer.
